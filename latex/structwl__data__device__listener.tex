The data\+\_\+device.\+data\+\_\+offer and the data\+\_\+offer.\+offer events are sent out immediately before this event to introduce the data offer object. The selection event is sent to a client immediately before receiving keyboard focus and when a new selection is set while the client has keyboard focus. The data\+\_\+offer is valid until a new data\+\_\+offer or N\+U\+LL is received or until the client loses keyboard focus. The client must destroy the previous selection data\+\_\+offer, if any, upon receiving this event.
