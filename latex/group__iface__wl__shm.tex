\doxysubsection{Detailed Description}
A singleton global object that provides support for shared memory.

Clients can create wl\+\_\+shm\+\_\+pool objects using the create\+\_\+pool request.

At connection setup time, the wl\+\_\+shm object emits one or more format events to inform clients about the valid pixel formats that can be used for buffers.
