repeat rate and delay

Informs the client about the keyboard\textquotesingle{}s repeat rate and delay.

This event is sent as soon as the wl\+\_\+keyboard object has been created, and is guaranteed to be received by the client before any key press event.

Negative values for either rate or delay are illegal. A rate of zero will disable any repeating (regardless of the value of delay).

This event can be sent later on as well with a new value if necessary, so clients should continue listening for the event past the creation of wl\+\_\+keyboard. 
\begin{DoxyParams}{Parameters}
{\em rate} & the rate of repeating keys in characters per second \\
\hline
{\em delay} & delay in milliseconds since key down until repeating starts \\
\hline
\end{DoxyParams}
