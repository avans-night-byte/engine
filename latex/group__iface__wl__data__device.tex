\doxysubsection{Detailed Description}
There is one wl\+\_\+data\+\_\+device per seat which can be obtained from the global wl\+\_\+data\+\_\+device\+\_\+manager singleton.

A wl\+\_\+data\+\_\+device provides access to inter-\/client data transfer mechanisms such as copy-\/and-\/paste and drag-\/and-\/drop.
