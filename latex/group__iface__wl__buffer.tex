\doxysubsection{Detailed Description}
A buffer provides the content for a wl\+\_\+surface. Buffers are created through factory interfaces such as wl\+\_\+drm, wl\+\_\+shm or similar. It has a width and a height and can be attached to a wl\+\_\+surface, but the mechanism by which a client provides and updates the contents is defined by the buffer factory interface.
