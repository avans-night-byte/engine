\doxysubsection{Detailed Description}
The wl\+\_\+data\+\_\+source object is the source side of a wl\+\_\+data\+\_\+offer. It is created by the source client in a data transfer and provides a way to describe the offered data and a way to respond to requests to transfer the data.
