\doxysubsection{Detailed Description}
A wp\+\_\+relative\+\_\+pointer object is an extension to the wl\+\_\+pointer interface used for emitting relative pointer events. It shares the same focus as wl\+\_\+pointer objects of the same seat and will only emit events when it has focus.
