configure the popup surface

This event asks the popup surface to configure itself given the configuration. The configured state should not be applied immediately. See xdg\+\_\+surface.\+configure for details.

The x and y arguments represent the position the popup was placed at given the xdg\+\_\+positioner rule, relative to the upper left corner of the window geometry of the parent surface.
