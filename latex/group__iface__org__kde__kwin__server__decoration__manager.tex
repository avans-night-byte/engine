\doxysubsection{Detailed Description}
This interface allows to coordinate whether the server should create a server-\/side window decoration around a wl\+\_\+surface representing a shell surface (wl\+\_\+shell\+\_\+surface or similar). By announcing support for this interface the server indicates that it supports server side decorations.
