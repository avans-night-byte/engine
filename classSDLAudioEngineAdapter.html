SDLAudioEngineAdapter::SDLAudioEngineAdapter ( )

A class that holds all the needed functions to load in and play music

You can either choose to load a sound file directly from a file path or trough memory by adding it to a dictionary

Some functions are declared static for easier sound managing and optimisation. Only loading and playing of a new file requires an instance of the object Constructor initializes the Mix_OpenAudio en Mix_init once so it can be used later.
